AFSC NOAA’s JavaScript version of EcoDAAT Data Management system is based on ESRI’s WebAppBuilder. 

Custom configuration can be found here:

https://doc.arcgis.com/en/web-appbuilder/create-apps/widget-overview.htm

Contact kimberly.bahl@noaa.gov and jan.benson@noaa.gov for more information and troubleshooting.

Updated Spring of 2018

 

 

Login using your LDAP NOAA email and password

 

Querying the EcoDAAT database can be done through the spatial search or a text search.  Users can also select points to find information at that location

 

Spatial Search Results are shown on the map and the table

 

Text Search has numerous filtering options.  Each table or layer has different optional query values

 

Searching can be filtered by different Date options

 

The output fields can be changed under the Choose Fields tab

 

Query results are listed and in the table, you can select different values or export out to CSV.  The table also has a number of other tools for selection

 

Additional tools can be found in the header of the site.  Measuring, drawing and other tools are available

 

The Layer List is available for selecting different layers and under each layer is the legend for each table

 

Under tools, users can select to export into different GIS formats

 

The TimeSlider can be used for a quick visualization of the data by different time extents

 

 

The project includes a number of different folders and files (html, js, css)

 

The config_eSearch.json file can be used to edit different field options and url strings for the application

 

The main search and query widget is in the eSearch folder.  The widget.js and widget.html can be edited to update functionality and text labels

 

The strings.js file has additional configuration for text and alert windows
